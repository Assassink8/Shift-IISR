Edge Injection Ablation Commands
================================

Notes
-----
- Replace /path/to/lr_images with the LR test image folder.
- Replace <RUN_DIR> with the timestamped training output directory, for example:
  /share/huayunpeng-local/checkpoints/sobel_baseline/2026-xx-xx-xx-xx
- Replace checkpoint_step=18000 if the experiment uses another saved step.
- Defaults are:
  diffusion.params.edge_operator=sobel
  diffusion.params.edge_normalize=True
  diffusion.params.edge_strength=1.0


1. Baseline: Sobel + Normalized + Strength 1.0
------------------------------------------------

Train:

CUDA_VISIBLE_DEVICES=0 python main.py \
  --cfg_path configs/bicx4_swinunet_lpips.yaml \
  --save_dir /share/huayunpeng-local/checkpoints/sobel_baseline

Inference:

CUDA_VISIBLE_DEVICES=0 python inference_resshift.py \
  -i /path/to/lr_images \
  -o result_sobel_baseline \
  --task bicsr --scale 4 \
  --cfg_path configs/bicx4_swinunet_lpips_infer.yaml \
  checkpoint_path=<RUN_DIR>/ckpts \
  checkpoint_step=18000
